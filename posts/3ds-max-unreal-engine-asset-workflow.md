## Introduction

From the get go we will jump straight to the point. In the last couple of years **Unreal Engine** has started creeping into the ArchViz scene. Nowadays, it's almost like the next industry standard.

With the new standard, we have a new set of challenges we need to address. This tutorial will try and cover the basics you will need to know in order to create assets for UE4.

I divided this into five videos so it's easier to follow.

## Phase 1: Modeling The Low Polygon Model

When you are creating a model, you can either start from a high poly model or a low poly one. The main difference is that if you start from a high poly, you will need to do a **retopology** run.

In this case, since we didn't have any sculpting, we start from scratch. In the first video you will see how to logically divide the model and then start modeling the base.
